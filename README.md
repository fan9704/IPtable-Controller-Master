# Iptables Controller Master

---

## Get Started

----

### Docker-Compose(Recommended)

```shell
docker-compose up -d
```

----

### Self-Host

#### Require

- OpenJDK/JRE 17
- Maven 3 
- RabbitMQ
- Elasticsearch
- Kibana (Optional)

#### Run Server

```shell
mvn spring-boot:run
```

---

## After Host IPC-Master 

### Run Client System

[IPtable-Controller-Backend](https://github.com/fan9704/IPtable-Controller-Backend)
